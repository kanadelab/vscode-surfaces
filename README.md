# surfaces.txtエディタエクステンション
伺かゴースト用のsurfaces.txt編集をサポートする言語拡張のエクステンションです。

## やってくれること
* シンタックスハイライト
* マウスホバーによるかんたんな説明と、設定内容の簡易表示
* 入力候補の自動表示
* elementやanimationを記入する際の記入内容ヘルプ
